🚓爬取政府資料開放平台：警察資料

以下是使用 Selenium 所撰寫的 Python 爬蟲，從 政府資料開放平台 搜尋「警察」，並將前 3 頁的資料標題與說明爬下來，輸出為 data.csv。

  1. ✅ 準備好 Selenium + ChromeDriver 的環境。
  2. ✅ 使用 webdriver_manager 幫忙自動下載對應的驅動版本，避免手動安裝。
  3. ✅ 匯入處理瀏覽器元素、等待網頁載入、錯誤處理等必要模組（如 By、WebDriverWait、EC 等）。
  4. ✅ 準備後續功能：寫爬蟲流程、抓取資料、儲存為表格（CSV 檔）。


from selenium import webdriver 
from selenium.webdriver.chrome.service import Service
from selenium.webdriver.common.by import By
from selenium.webdriver.common.keys import Keys
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC
import time
import pandas as pd
from webdriver_manager.chrome import ChromeDriverManager
from selenium.common.exceptions import NoSuchElementException, ElementNotInteractableException

設定 Chrome 選項，啟動瀏覽器

								
options = webdriver.ChromeOptions()
options.add_argument("--disable-blink-features=AutomationControlled")
driver = webdriver.Chrome(service=Service(ChromeDriverManager().install()), options=options)
driver.get("https://data.gov.tw/")
driver.maximize_window()

搜尋關鍵字：警察


try:
    search_box = WebDriverWait(driver, 10).until(
        EC.presence_of_element_located((By.ID, "searchbar-input"))
    )
    search_box.send_keys("警察")
    search_box.send_keys(Keys.ENTER)
except NoSuchElementException:
    print("❌ 無法找到搜尋框")
    driver.quit()
    exit()

儲存資料列表，抓取當頁資料


data_list = []

def extract_page_data():
    try:
        WebDriverWait(driver, 10).until(
            EC.presence_of_all_elements_located((By.CLASS_NAME, "dataset-item-left"))
        )
        items = driver.find_elements(By.CLASS_NAME, "dataset-item-left")
        for item in items:
            try:
                title = item.find_element(By.CLASS_NAME, "dataset-title").text.strip()
                reftitle = item.find_element(By.CLASS_NAME, "dataset-content").text.strip()
                data_list.append({"標題": title, "說明": reftitle})
                print(f"✅ {title} ➡️ {reftitle}")
            except NoSuchElementException:
                continue
    except Exception as e:
        print("❌ 抓取資料失敗：", e)

爬取前 3 頁


count = 0
while count < 3:
    extract_page_data()
    try:
        WebDriverWait(driver, 10).until(
            EC.invisibility_of_element_located((By.CLASS_NAME, "el-loading-mask"))
        )
        next_button = WebDriverWait(driver, 10).until(
            EC.element_to_be_clickable((By.CSS_SELECTOR, ".btn-next"))
        )
        driver.execute_script("arguments[0].click();", next_button)
        time.sleep(2)
    except (NoSuchElementException, ElementNotInteractableException):
        print("⚠️ 沒有下一頁")
        break
    count += 1

driver.quit()
